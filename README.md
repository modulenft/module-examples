## Getting Started

1. Go to [dash.modulenft.xyz](https://dash.modulenft.xyz) and create an account.
2. Create a new app.
3. Copy the API key.
4. Clone this repo.
5. `cd` into the example you want to run.
6. Create/edit a `.env` file and add your API key and other configuration options.
